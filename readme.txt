== Description ==

This plugin adds a prettyPhoto Gallery Lightbox plugin for every gallery created on wp editor. 

Any errors or bugs you find please use the support forum.

If you find this plugin helpful, we appreciate if you give us your review. 

== Installation ==

1. Upload wp-gallery-lightbox folder to the `/wp-content/plugins/` directory or use plugin adin page to upload th zip file.
2. Activate the plugin through the 'Plugins' menu in WordPress
